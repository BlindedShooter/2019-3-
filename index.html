정자꽃뫼 중고등부 2019 고3 졸업 여행

“함께 가자 이 길을”

  • 일시: 2019년 2월 15일 ~ 16일
  • 장소: 전주한옥마을 일대
  • 취지: 중고등부 주일학교의 마무리와 올바른 성인으로서의 출발
  • 목적: 삶의 방향성 수립 및 올바른 신앙인으로써의 제고, 청년회 소속 청년들과의 관계 형성
  • 준비물: 세면도구, 편한 복장

한옥마을 추천 관광코스

링크: 전주시청 (모바일에서 느릴 수 있음)

1. 한옥마을 단숨에 보는 코스 (1~2시간)

①전동성당 → ②경기전 → ③교동미술관 → ④최명희문학관 → ⑤부채문화관 → ⑥한옥마을역사관 → ⑦태조로 사거리 → ⑧오목대

2. 골목과 체험이 있는 코스 (2~3시간)


①오목대 → 태조로 → ②전동성당 → ③경기전 → ④교동미술관 → ⑤최명희문학관(체험) → ⑥부채문화관(체험) → ⑦한옥마을역사관 → ⑧태조로 → ⑨전통한지원(체험) → ⑩승광재 → ⑪전통술박물관(체험) → ⑫소리문화관(체험) → ⑬전주김치문화관(체험)

3. 산책과 사색이 있는 코스 (2~3시간)


①풍남문 → ②전동성당 → ③경기전 → ④오목대 → ⑤자만벽화마을 → ⑥전주향교 → ⑦완판본문화관 → ⑧오목교(다리) → ⑨국립무형유산원 → ⑩남천교(다리)

이외에도 역사의 숨결을 느끼는 코스 (2~3시간), 슬로투어코스 (3시간 이상), 역사탐방코스 (2시간 이상) 등 코스가 있으니 관심 있는 분들은 위의 링크에서 확인하고, 꼭 정해진 코스가 아니어도 괜찮으니 자유롭게 여행하세요!

한옥마을 관광 10대 명소
- 한옥마을 명소 10곳 소개와 간단한 설명.

한옥마을 포토존
- 사진 예쁘게 나오는 곳들이 예제 사진과 함께 소개.

함께 가자 이 길을

기차 안에서 세운 여행계획대로 여행하면 됩니다. 한옥마을의 아름다운 길을 가볍게 걸으며, 서로가 걸어온 길과 앞으로 가야할 길에 대해 대화하는 시간이 되었으면 합니다.

  • 반드시 1시간 마다 인솔교사들에게 인증샷과 함께 위치를 알려주시기 바랍니다!!

미션

  • 다른 조를 발견 시 도촬에 성공하면 선물이 있습니다!
  • 재미있는 포즈로 사진 촬영 성공 시 선물이 있습니다!

사진 경연대회

여행 동안 찍은 재미있는 사진으로 경연대회를 합니다! 투표를 가장 많이 받은 팀에게는 선물이 있습니다!

한옥마을 화장실 위치
- 네이버 지도에 화장실들이 마커로 찍혀있음. 마커 클릭시 화장실 정보 나옴.

한옥마을 관광안내소 위치
- 네이버 지도에 관광안내소 위치를 마커에 찍어줌. 화장실, 셔틀버스, 공영자전거 대여소 위치도 확인할 수 있음.

한옥마을 문화시설 안내
- 한옥마을의 각종 문화/체험시설들의 정보 제공.
