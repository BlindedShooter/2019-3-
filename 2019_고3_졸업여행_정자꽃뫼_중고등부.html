한옥마을 추천 관광코스

링크: 전주시청 (모바일에서 느릴 수 있음)

1. 한옥마을 단숨에 보는 코스 (1~2시간)

①전동성당 → ②경기전 → ③교동미술관 → ④최명희문학관 → ⑤부채문화관 → ⑥한옥마을역사관 → ⑦태조로 사거리 → ⑧오목대

2. 골목과 체험이 있는 코스 (2~3시간)


①오목대 → 태조로 → ②전동성당 → ③경기전 → ④교동미술관 → ⑤최명희문학관(체험) → ⑥부채문화관(체험) → ⑦한옥마을역사관 → ⑧태조로 → ⑨전통한지원(체험) → ⑩승광재 → ⑪전통술박물관(체험) → ⑫소리문화관(체험) → ⑬전주김치문화관(체험)

3. 산책과 사색이 있는 코스 (2~3시간)


①풍남문 → ②전동성당 → ③경기전 → ④오목대 → ⑤자만벽화마을 → ⑥전주향교 → ⑦완판본문화관 → ⑧오목교(다리) → ⑨국립무형유산원 → ⑩남천교(다리)

이외에도 역사의 숨결을 느끼는 코스 (2~3시간), 슬로투어코스 (3시간 이상), 역사탐방코스 (2시간 이상) 등 코스가 있으니 관심있으신 분들은 위의 링크에서 확인해주세요!

유용한 링크

한옥마을 관광 10대 명소
- 한옥마을 명소 10곳 소개와 간단한 설명.

한옥마을 포토존
- 사진 예쁘게 나오는 곳들이 예제 사진과 함께 소개.

한옥마을 화장실 위치
- 네이버 지도에 화장실들이 마커로 찍혀있음. 마커 클릭시 화장실 정보 나옴.

한옥마을 관광안내소 위치
- 네이버 지도에 관광안내소 위치를 마커에 찍어줌. 화장실, 셔틀버스, 공영자전거 대여소 위치도 확인할 수 있음.

한옥마을 문화시설 안내
- 한옥마을의 각종 문화/체험시설들의 정보 제공.
